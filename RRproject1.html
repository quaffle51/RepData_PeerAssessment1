Imputing missing values

There are a number of days/intervals where there are missing values (coded as NA). The presence of missing days may introduce bias into some calculations or summaries of the data.

1. Calculate and report the total number of missing values in the dataset (i.e.the total number of rows with NAs)

missing <- sqldf("SELECT count(*) FROM df WHERE steps is NULL")

The number of missing values in the data set is 2304.

2. Devise a strategy for filling in all of the missing values in the dataset.

The strategy does not need to be sophisticated. For example, you could use the mean/median for that day, or the mean for that 5-minute interval, etc.

My strategy is as follows. a. If the interval is between 0 and 499, steps <- mean of steps between intervals 0 and 499 for all days. b. If the interval is between 500 and 999, steps <- mean of steps between intervals 500 and 999 for all days. c. If the interval is between 1000 and 1499, steps <- mean of steps between intervals 1000 and 1499 for all days. d. If the interval is between 1500 and 1999, steps <- mean of steps between intervals 1500 and 1999 for all days. e. if the interval is between 2000 ane 2499, steps <- mean of steps between intervals 2000 ane 2499 for all days.

This strategy can be achieved through the following code.

a <- sqldf("SELECT round(avg(steps),0) as avg FROM dfcc WHERE interval between    0 and  499")
b <- sqldf("SELECT round(avg(steps),0) as avg FROM dfcc WHERE interval between  500 and  999")
c <- sqldf("SELECT round(avg(steps),0) as avg FROM dfcc WHERE interval between 1000 and 1499")
d <- sqldf("SELECT round(avg(steps),0) as avg FROM dfcc WHERE interval between 1500 and 1999")
e <- sqldf("SELECT round(avg(steps),0) as avg FROM dfcc WHERE interval between 2000 and 2499")
sapply(c(a,b,c,d,e), function(index) index)
## avg avg avg avg avg 
##   1  69  44  58  11
